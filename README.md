## アジェンダ

- SQLの習得
    - 教材は「[スッキリわかるSQL入門](https://www.amazon.co.jp/dp/4295018465)」
    - わからなくても、学習ペースをおとさない。最低限、教材を写経すれば良しとする。
    - １周目で概略を把握、２周目で詳細を理解するぐらいのつもりで。
- Gitの使い方を習得
    - 学習した内容はSQL文含め、markdownで記述してGitHubで共有する 
    - Gitのツールは何を使ってもOK
- マークダウンに慣れる
- VScodeに慣れる
